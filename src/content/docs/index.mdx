<div class="hero-code">

```sh frame="none"
$ envsh login
$ envsh push .env --project my-api --env production
$ envsh pull production --project my-api
$ envsh run production --project my-api -- node server.js
```

</div>

<div class="feature-row">

**Zero-knowledge** — the server can't read your secrets. Not "won't" — *can't*.

**Your SSH keys** — no new credentials. Uses Ed25519 keys you already have.

**Push & pull** — `push` encrypts and uploads. `pull` decrypts and downloads.

**CI/CD ready** — machine identities scoped to one project, one environment.

</div>
