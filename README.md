# envsh.dev

Documentation website for [envsh](https://github.com/envshq/envsh) — zero-knowledge secret management for teams.

Built with [Astro Starlight](https://starlight.astro.build/). Deployed to GitHub Pages at [envsh.dev](https://envsh.dev).

## Development

```bash
pnpm install
pnpm dev        # localhost:4321
pnpm build      # production build → ./dist/
```

## Deployment

Pushes to `main` auto-deploy via GitHub Actions to GitHub Pages.
